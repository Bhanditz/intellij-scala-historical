Replaces filter/filterNot on a Set with intersect or diff if possible.

set.filter(others.contains(_)) -> set.intersect(others)

set.filterNot(others.contains)) -> set.diff(others)

set.filter(x => !others.contains(x)) -> set.diff(others)

set.filterNot(x => !others.contains(x)) -> set.intersect(others)
